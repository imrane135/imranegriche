Face Recognition Project

Project Description

This project implements a face recognition system that can identify and verify individuals from video streams.
The system leverages deep learning-based face detection and recognition models to analyze facial features and compare them against a database of known individuals it gives also the option to add new faces (sign up).
The project integrates a MySQL database to store and manage facial embeddings, user details, and authentication logs. The system can be used for various applications, such as attendance tracking, security authentication, and access control.

Technologies Used
Python – Main programming language for development
OpenCV – Face detection and image preprocessing
DeepFace – Pre-trained deep learning model for face recognition
MySQL – Database to store user information and facial embeddings
Flask – Web framework for API and user interface
PyMySQL – Database connection and queries
PyQt – GUI for user interaction
