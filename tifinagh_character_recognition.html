Tifinagh

Project Description

This project is an AI-powered optical character recognition (OCR) system designed to identify and process Tifinagh characters from images and scanned documents. It leverages deep learning models to extract and recognize characters, enabling applications such as text digitization, language preservation, and historical manuscript analysis.

To achieve high accuracy, we experimented with multiple approaches, including Tesseract OCR, Support Vector Machines (SVM), PaddleOCR, and Convolutional Neural Networks (CNN). While SVM, PaddleOCR, and CNN delivered promising results, SVM struggled with performance on the test dataset. Ultimately, our final models were built using CNN and PaddleOCR, which provided the best accuracy for Tifinagh script recognition.

The system is implemented using Flask, providing a simple and efficient interface for processing and recognizing Tifinagh characters. Users can upload images, and the model processes them to extract and display recognized text in real time.
