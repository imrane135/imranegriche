Tifinagh

Project Description

Tifinag considered as an ofiicial languagen in Morocoo, so the objective of this project was to create a model capable of converting image to Text using Optical Character Recognition there are alot of website that can do that in many languages like English, Freanch and Arabic but threre is no website until this day that can convert image to text in Tifinagh language

at this project we have tried many approches such as Tesseract, Support Vector Machine, PaddleOCR and Convolutinal Nearal Networks (CNN) we found good result in SVM, paddle and CNN but in SVM our model did not give a good results on the test so thats why our final models were based on CNN and PaddleOCR
