Project Description

This project focuses on correcting user input and generating emails in different formats, both formal and informal. You can also specify the tone or emotion of the email.

The project utilizes prompt engineering as an alternative to creating your own language model from scratch or fine-tuning a model, which can be time-consuming and require significant computational resources. Prompt engineering offers a simpler and more efficient approach.
