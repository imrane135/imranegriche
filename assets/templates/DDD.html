About the Project

This project focuses on detecting dermatological diseases using deep learning and computer vision techniques. By employing a convolutional neural network (CNN), the system is trained to identify and classify skin lesions into benign and malignant categories. The model was trained using the HAM10000 dataset, which includes dermatoscopic images for disease diagnosis.

The main goal of the project is to help healthcare professionals in early detection of conditions like melanoma, basal cell carcinoma, and other skin diseases through image analysis.

Key Features

  • Classify skin lesions as benign or malignant
  • Deep learning model for high accuracy in prediction
  • Use of convolutional neural networks (CNN)
  • Support for dermatoscopic image analysis
